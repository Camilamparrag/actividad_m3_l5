¿Cuál estructura te pareció más flexible?

La lista, porque puedes agregar, eliminar y modificar elementos facilmente. Es como la más versatil para trabajar.

¿Para qué tipo de problema usarías un diccionario en lugar de una lista?

Usaría un diccionario cuando necesito guardar datos con un significado, por ejemplo nombre, edad, ciudad, etc. Es más ordenado porque cada dato tiene su clave, no como la lista que es solo por posición.
